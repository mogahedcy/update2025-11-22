تحسين محركات البحث (SEO)
أولوية قصوى
التقييم: 4/10 - يحتاج تحسين عاجل

المشاكل الحرجة
Keyword Stuffing: تكرار "محترفين الديار" و"جدة" بشكل مبالغ فيه يضر بالترتيب
Meta Tags: عدم وجود meta description فريد لكل صفحة
العنوان الرئيسي: طويل جداً (يجب أن يكون أقل من 60 حرف)
البنية الهيكلية: استخدام عشوائي لـ H1, H2, H3
Schema Markup: غير موجود (مهم للأعمال المحلية)
خطة العمل التفصيلية
1. تحسين Meta Tags (أسبوع واحد)
العنوان المقترح:

محترفين الديار | مظلات وبرجولات جدة - ضمان 10 سنوات
الوصف المقترح:

تركيب مظلات، برجولات، سواتر في جدة بخبرة +15 سنة. ضمان 10 سنوات، خدمة 24/7. اتصل الآن: 0553719009
2. تقليل Keyword Stuffing (أسبوعين)
قبل: "محترفين الديار - خدمات محترفين الديار في جدة - محترفين الديار للمظلات"

بعد: "خدماتنا المتخصصة في جدة - حلول المظلات والبرجولات"

استخدام المرادفات والضمائر بدلاً من التكرار المباشر

3. إضافة Schema Markup (3 أيام)
Schema Types المطلوبة:

• LocalBusiness Schema
• Product Schema لكل خدمة
• Review Schema للشهادات
• FAQPage Schema
{ "@context": "https://schema.org", "@type": "LocalBusiness", "name": "محترفين الديار العالمية", "image": "https://aldeyarksa.tech/logo.jpg", "address": { "@type": "PostalAddress", "addressLocality": "جدة", "addressCountry": "SA" }, "telephone": "+966553719009", "priceRange": "$$" }
4. تحسين البنية الهيكلية (أسبوع واحد)
القواعد:

• H1 واحد فقط لكل صفحة (العنوان الرئيسي)
• H2 لعناوين الأقسام الرئيسية
• H3 للعناوين الفرعية
• تسلسل منطقي بدون قفزات (H1 → H2 → H3)
5. تحسين الصور (أسبوع واحد)
• إضافة alt text وصفي لكل صورة
• استخدام أسماء ملفات وصفية (مظلات-سيارات-جدة.webp)
• ضغط الصور بدون فقدان جودة
• استخدام WebP format
6. Internal Linking Strategy (أسبوعين)
• ربط الخدمات ذات الصلة ببعضها
• إضافة breadcrumbs للتنقل
• استخدام anchor text وصفي
• إنشاء sitemap.xml
النتائج المتوقعة بعد التحسين
زيادة الظهور في نتائج البحث المحلية بنسبة 40-60%
تحسين Click-Through Rate (CTR) بنسبة 25-35%
ظهور Rich Snippets في جوجل (نجوم التقييم، العنوان، الهاتف)
تحسين الترتيب للكلمات المفتاحية الرئيسية في 3-6 أشه
