# دليل تحسين Alt Text للصور - SEO

## ما هو Alt Text؟
Alt Text (النص البديل) هو وصف نصي للصورة يساعد:
- محركات البحث في فهم محتوى الصورة
- المستخدمين ذوي الإعاقة البصرية عبر قارئات الشاشة
- عرض النص عند فشل تحميل الصورة

## قواعد كتابة Alt Text محسّن للـ SEO

### ✅ افعل:
1. **كن وصفياً ومحدداً** - اوصف ما في الصورة بدقة
2. **استخدم الكلمات المفتاحية بشكل طبيعي** - ضمّن كلمات مفتاحية ذات صلة
3. **اجعله موجزاً** - 125 حرف أو أقل (تقريباً 10-15 كلمة)
4. **ضمّن الموقع** - اذكر "جدة" أو "السعودية" عند الملاءمة
5. **كن محدداً للسياق** - اربط الوصف بمحتوى الصفحة

### ❌ لا تفعل:
1. لا تبدأ بـ "صورة لـ" أو "صورة تظهر"
2. لا تحشُ بالكلمات المفتاحية
3. لا تترك Alt Text فارغاً للصور المهمة
4. لا تستخدم نفس النص لصور مختلفة

## قائمة التحقق النهائية ✓

قبل نشر أي صورة، تأكد من:
- [ ] Alt Text وصفي ومحدد
- [ ] يتضمن كلمة مفتاحية ذات صلة بشكل طبيعي
- [ ] يذكر "جدة" أو الموقع عند الملاءمة
- [ ] طوله مناسب (10-15 كلمة)
- [ ] لا يبدأ بـ "صورة" أو "صورة لـ"
- [ ] فريد ولا يتكرر مع صور أخرى
- [ ] يساعد المستخدمين ذوي الإعاقة البصرية
- [ ] يعزز SEO للصفحة

## الخلاصة

Alt Text الجيد:
1. **يصف** محتوى الصورة بدقة
2. **يتضمن** كلمات مفتاحية بشكل طبيعي
3. **يذكر** الموقع والتفاصيل المهمة
4. **يساعد** المستخدمين ومحركات البحث معاً

**تذكّر**: Alt Text هو فرصة SEO ثمينة - استخدمها بذكاء!
